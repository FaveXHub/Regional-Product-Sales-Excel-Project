# Regional-Product-Sales

Sales Performance Dashboard (Excel Analytics Project)

This project analyzes a retail sales dataset using Excel PivotTables, PivotCharts, and Slicers to create an interactive dashboard for business insights. The goal was to transform raw data into a visual report that helps identify trends, top-performing products, revenue distribution, and customer purchasing behavior across different regions.


---

##Project Objectives

Convert raw sales data into a structured Excel table

Analyze revenue, quantity sold, and product performance

Build interactive PivotCharts and KPI cards

Enable dynamic filtering with slicers (Region, Category, Date)

Design a clean, user-friendly sales dashboard



---

##Tools & Skills Used

Microsoft Excel

PivotTables & PivotCharts

Data Cleaning

Slicers & Dashboard Interactivity

Data Visualization & KPI Reporting



---

##Process Overview

1. Cleaned and structured the dataset


2. Built a master PivotTable for analysis


3. Duplicated pivots for multiple perspectives


4. Created visualizations for revenue, trends, and categories


5. Added slicers and connected all visuals


6. Designed and formatted the final dashboard layout




---

##Dashboard Features

Total Revenue & Quantity KPI cards

Sales trend over time

Revenue by region

Top product categories and best-selling items

Top Region Manager Performer

Fully interactive slicers for real-time filtering



---

##Key Insights

Laptops, Tablets and Printers generated the highest revenue

Region North showed the strongest overall sales fir the past 3 years

Sales peaked in Q1

Laptop is the top performer in both revenue and quantity


---

##How to Use

1. Download and open the Excel file


2. Use slicers (Region, Category, Date) to filter the dashboard


3. Hover over charts for detailed values


4. Explore patterns and trends dynamically
